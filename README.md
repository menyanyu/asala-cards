# 阿萨拉扑克牌收集

三角洲行动 S10 裂变赛季 · 阿萨拉扑克牌收集追踪工具

## 如何下载到电脑

点击右上角绿色的 **Code ▾** 按钮 → **Download ZIP**

下载后解压，双击 `asala-cards.html` 即可使用。

## 使用方法

1. 双击打开 `asala-cards.html`（用浏览器）
2. 每张牌初始都是**灰色未获得**
3. 在游戏中捡到某张牌 → **点一下**牌面，亮起彩色光效 = 已获得
4. 不小心点错了 → **再点一下**，变回灰色 = 取消获得
5. 牌盒也能点，获得后发红光
6. 进度自动保存在浏览器里，关了再打开不会丢
7. 底部「重置收集进度」可一键清空重来

| 品质 | 边框光效 | 包含 |

| 金色 | 🟡 | 小王 |
